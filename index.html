GLM-5: from Vibe Coding to Agentic Engineering

GLM-5 Team · Zhipu AI & Tsinghua University

GLM-5 是 744B 总参 / 40B 激活 的下一代 MoE 旗舰，首个 AA Intelligence Index v4.0 拿到 50 的开源模型。技术四件套：(1) MLA + Muon Split + MLA-256 让 latent attention 追平 GQA-8 同时降低 decoding 成本；(2) 共享参数 3 层 MTP draft 模型 acceptance 长度 2.76 高于 DeepSeek-V3.2 的 2.55；(3) 在 base 上做 DSA 持续预训练，仅 20B token 即把 sparse attention 在长文本上无损切换；(4) 全异步 RL（slime）+ TITO Gateway + 双侧 importance sampling + 跨阶段在线蒸馏，让 100k 级 long-horizon agent rollout 不阻塞训练。SWE-Bench Verified 77.8、Vending-Bench 2 一年余额 4,432 USD。

MiMo-V2-Flash Technical Report

LLM-Core Xiaomi

MiMo-V2-Flash 是 309B 总参 / 15B 激活的 MoE 模型，三大支柱：(1) 5:1 Hybrid Sliding Window Attention（窗口仅 128 + 学习型 attention sink），KV 缓存与注意力计算降至约 1/6；(2) 轻量化 Multi-Token Prediction 既作训练目标也作 self-speculative draft，3 层 MTP 实测 2.6× 解码加速；(3) MOPD（Multi-Teacher On-Policy Distillation）把多领域 RL 教师的 token 级 reverse-KL 与 ORM 优势相加，缓解多技能跷跷板。SWE-Bench Multilingual 71.7 居开源第一。

DeepSeek-AI

Native Sparse Attention: Hardware-Aligned and Natively Trainable Sparse Attention

DeepSeek-AI · Peking University · University of Washington

NSA 是一个原生可训练的稀疏注意力架构——sparsity 从预训练第一步就参与梯度，而不是事后剪枝。三分支设计：(1) Token Compression 用 MLP 把连续 KV block 压成摘要向量做粗粒度全局视野；(2) Token Selection 直接复用 cmp 分支的 softmax 分数当 importance score 挑 top-n 块——零额外打分开销且全程可微；(3) Sliding Window 独立分支专管局部模式防短路。配套 GQA-grouped Triton kernel 把外层切成 GQA group 让组内 head 共享 KV 索引。在 27B/3B GQA-MoE 上预训练 270B token，9 项通用 benchmark 中 7 项反超 Full Attention，64K NIAH 满分；64K 序列上 9.0× 前向 / 6.0× 反向 / 11.6× 解码加速。
